Main Description

Se você estiver considerando o uso de uma planilha para acompanhar o Project Backlog e o Project Burndown considere estes dois modelos gentilmente fornecidos por Clay Ver Valen e Petri Heiramo. Ambos os Templates são excelentes e, com certeza, um poderá ser usado em seu projeto. O Template de Petri usa macros que proporcionam uma rica funcionalidade, mas pode ser um impedimento para alguns. O Template de Petri contém uma amostra de dados, enquanto o de Clay fornece um exemplo em separado.
